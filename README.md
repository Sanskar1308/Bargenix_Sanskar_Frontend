## 🚀 Quick Start

1. Clone the repository
```bash
git clone https://github.com/Sanskar1308/Bargenix_Sanskar_Frontend.git
```

2. Install dependencies
```bash
cd Bargenix_Sanskar_Frontend
npm install
```

3. Start development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```
